  • BUCKET'S NEW FRIEND

    Designing and coding a browser game. In this theme our only assignment was to design and develop a simple game point and click game using HTML, CSS and Javascript. With the game weâ€™ve been told to create a story and characters that we could ten use inside of our game. I decided to create game around the idea that as a main character you found a raccoon and decided to keep him as a pet. The challenge in game is to collect hearts so your new pet will bound with Yahoo your dog. I wanted to make the game as relaxing as possible, with both comforting design and music. During this assignment I designed my first game interface, all the graphics in the project needed to be vectors and that also created some challenges for me. During the process of designing the game I created couple of different versions of the interface before I committed to the final one. It was very important for me for me to tell the story with my game and make all the elements, the story, the characters as well as UI elements and music to be a part of one uniform design. When I had my prototype ready I could start working on my code which was the hardest part of the assignment as we just got introduced for the first time to Javascript. I learned how to write a code to create my desired gaming experience and add animations using Javascript.

  • CODE AND DESIGN

    Tools used in project include: HTML, CSS, JavaScript, Adobe Suite (photoshop, illustrator, XD)

  • SKILLS LEARNED DURING THIS ASSIGNMENT

    Character design, proportions, background design, gamification, storytelling, tracing images in illustrator, HTML (Combinator, selectors, pseudo-selectors), CSS( ease-in, ease-out, transforms, animations, keyframes, sprite animation,advanced positioning, DOM) Javascript( events, functions, selecting and controlling media, if else statements, Boolean variables), State machine diagram, activity diagram
