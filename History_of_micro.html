  • HISTORY OF THE MICROCOMPUTER

    Durning this assignment I had to create first website with multiple pages. Content I used on these pages was an article about history of the micro-computers and how they worked. Part of this assignment was to write an essay about my history of using computers and adding it to the webpage. This time we didn’t get photos that we could use so as part of the design process we needed to find/make copyright free photos that we could use on our domain. Main challenge of this project was creating first webpage using css grids and responsive design. Also on this website we had to include navigation, I’ve learned how to use list element to create navigation and how to style in css to reach desired effect on my website. To make sure our websites are working well we were also introduced to process of validating the code.

  • CODE AND DESIGN

    Tools used in project include: HTML, CSS, Adobe Suite (Photoshop, XD)

  • SKILLS LEARNED DURING THIS ASSIGNMENT

    HTML elements (nav, aside, ul, ol, li, div, abbr), CSS elements (grid, using breakpoints, @media queries/syntax/types, flexbox), responsive design, mobile design, absolute links, relative links, internal links, adding fonts to the project, wireframes, interactive prototyping
