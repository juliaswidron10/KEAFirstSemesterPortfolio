  • RUSSIAN CONSTRUCTIVISM

    In this theme I made my first ever webpage using HTML and CSS. We had to create website to present the article about Russian Constructivism, the biggest challenge during working on this project was creating the most suitable website that would complement the article’s theme. We’ve been given photos that we could use during the assignment, I learned how to color grade photos so all the photos would have the same temperature and color scale to achieve more uniform design of the webpage

  • CODE AND DESIGN

    Tools used in project include: HTML, CSS, Adobe Photoshop, Illustrator, XD

  • SKILLS LEARNED DURING THIS ASSIGNMENT

    HTML elements (html skeleton, main, h1,2,3, p, a, section, img, figure, footer, strong, q,blockquote,cite) CSS ( styling texts, styling images, using class and id to style specific elements), gestalt laws, composition, typography, contrast, using color theory, alternative text, HEX, RGB colors, XD prototype, photo editing, image size/formats/assets, file structure, Fillezilla, photoshop, XD
